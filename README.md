# 📜 Policy Pages

Đây là kho lưu trữ chứa các trang pháp lý và chính sách chuẩn cho ứng dụng hoặc website của bạn. Dự án được thiết kế đơn giản, tập trung vào trải nghiệm đọc và dễ dàng tích hợp.

---

## 📂 Cấu trúc dự án

*   **`index.html`**: Trang chủ giới thiệu các chính sách.
*   **`privacy-policy.html`**: Chính sách bảo mật thông tin người dùng.
*   **`terms-of-use.html`**: Điều khoản sử dụng dịch vụ.
*   **`delete-account.html`**: Hướng dẫn và quy trình xóa tài khoản[cite: 14].
*   **`site.css`**: Tệp tin định dạng CSS chung giúp đồng bộ hóa giao diện cho tất cả các trang chính sách[cite: 14].

---

## 🛠️ Hướng dẫn tích hợp

1.  **Deploy**: Bạn có thể host trực tiếp các file này lên các nền tảng như Vercel, Netlify hoặc GitHub Pages.
2.  **Tùy chỉnh**: 
    *   Chỉnh sửa các nội dung văn bản trực tiếp trong từng file `.html`.
    *   Thay đổi phong cách thiết kế (màu sắc, font chữ) thông qua `site.css`.
3.  **Liên kết**: Đảm bảo điều hướng (navbar/footer) được liên kết đúng với tên các tệp tin HTML tương ứng.

---

## 📋 Ghi chú
*   Dự án sử dụng các tệp tin `.gitignore` để loại bỏ các file hệ thống không cần thiết như `.DS_Store` hay `Thumbs.db`[cite: 14].
